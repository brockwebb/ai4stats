# Chapter 5 Code Examples: Graph Thinking and Record Linkage

Run in order:

1. `01_household_network.py` — household network construction and metrics
2. `02_graph_types_visualization.py` — three graph type illustrations
3. `03_synthetic_records.py` — generate synthetic records for linkage
4. `04_blocking.py` — blocking strategies and pair reduction
5. `05_comparison_scoring.py` — comparison features and linkage classifier
6. `06_linkage_graph.py` — build and analyze the linkage graph
7. `07_survey_effects.py` — household clustering and geographic spillover
8. `08_exercise.py` — admin record linkage exercise and solution

## Requirements

Python 3.9+, numpy, pandas, matplotlib, scikit-learn, networkx
