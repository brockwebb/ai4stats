# Examples

This directory contains runnable code for each chapter of *AI for Official Statistics*.

The book chapters are prose-first — they explain concepts, framing, and judgment. The code here is the companion implementation: full, runnable notebooks and scripts that demonstrate the methods described in each chapter.

## How to Use

Each subdirectory corresponds to a chapter:

| Directory | Chapter |
|-----------|---------|
| `chapter-01/` | Regression and Classification for Survey Data |
| `chapter-02/` | Cross-Validation and Model Selection |
| `chapter-03/` | Decision Trees and Random Forests |
| `chapter-04/` | Neural Networks Basics |
| `chapter-05/` | Graph Thinking and Record Linkage |
| `chapter-06/` | Dimension Reduction and Geographic Segmentation |
| `chapter-07/` | Imputation Methods for Survey Data |
| `chapter-08/` | Bias, Fairness, and Equity in Federal AI/ML |
| `chapter-09/` | Synthetic Data Generation for Federal Statistics |
| `chapter-10/` | Statistical Disclosure Limitation in the Age of AI |
| `chapter-11/` | Transformers for Survey Text Classification |
| `chapter-12/` | Large Language Models for Survey Operations |
| `chapter-13/` | Agentic AI for Federal Statistical Operations |
| `chapter-14/` | Evaluating AI Systems for Federal Use |
| `chapter-15/` | Capstone: Reproducible AI-Assisted Research |

## Data

All examples use publicly available datasets:
- American Community Survey (ACS) Public Use Microdata Sample (PUMS)
- Census Bureau API
- Bureau of Labor Statistics data

No proprietary or restricted data is required.

## Running the Examples

Examples are provided as Jupyter notebooks (`.ipynb`) and Python scripts (`.py`). They can be run locally or in Google Colab.

To open in Colab: click the notebook file on GitHub, then click the "Open in Colab" badge at the top.
