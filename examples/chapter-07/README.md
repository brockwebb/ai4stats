# Chapter 7 Code Examples: Imputation Methods for Survey Data

Run in order:

1. `01_dataset_and_missingness.py` — generate data, introduce missingness, visualize mechanisms
2. `02_mean_imputation.py` — mean imputation and its failures
3. `03_conditional_mean.py` — conditional mean imputation by strata
4. `04_hot_deck.py` — hot-deck imputation with donor tracing
5. `05_regression_imputation.py` — deterministic and stochastic regression
6. `06_multiple_imputation.py` — multiple imputation and Rubin's rules
7. `07_rf_imputation.py` — Random Forest imputation
8. `08_diagnostics.py` — plausibility diagnostics and method comparison
9. `09_exercise.py` — hours_wk imputation exercise and solution

## Requirements

Python 3.9+, numpy, pandas, matplotlib, scikit-learn, scipy
